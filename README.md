# XFOIL_GUI
A simple Python GUI for running and visualizing XFOIL airfoil analysis.

Features
NACA and .dat airfoils
Reynolds number calculator
Runs XFOIL automatically
Plots: CL–α, CD polar, CM–α, L/D
Built-in console output
Requirements
Python 3.x
numpy, pandas, matplotlib
XFOIL executable
Run
python main.py
Note

Requires xfoil.exe to be installed or linked in the app.
